# FuelPro v1.0.0 - Installation Guide

## Quick Start

### Option 1: Install as PWA (Recommended - All Devices)
- **Android Chrome**: Visit https://3d3tjxc5r2qoc.kimi.page > Menu > "Add to Home Screen"
- **iOS Safari**: Visit https://3d3tjxc5r2qoc.kimi.page > Share > "Add to Home Screen"
- **Windows Chrome**: Visit https://3d3tjxc5r2qoc.kimi.page > Menu > "Install FuelPro"
- **Works offline** after first visit (service worker caches all assets)

### Option 2: Android APK
1. Download `FuelPro-v1.0.0.apk` to your Android phone
2. Enable "Unknown Sources" in Settings > Security
3. Open the APK file to install
4. Launch FuelPro from your app drawer

### Option 3: Windows (No Installation Required)
1. Download `FuelPro-Windows.zip` and extract
2. Double-click `FuelPro.bat` to launch in Chrome app mode
3. Or right-click `FuelPro.ps1` > "Run with PowerShell"

### Option 4: Use in Browser (Any Device)
Visit: https://3d3tjxc5r2qoc.kimi.page

## Features

- **Data Persistence**: All data saved in browser's IndexedDB - survives refresh, browser close, device restart

- **EPRA Prices**: Auto-synced Kenya fuel prices (Ksh 220.30/L petrol, Ksh 250.01/L diesel)
- **Location**: Auto-detects your exact GPS location for price accuracy
- **Document Center**: Upload files/folders with auto-categorization (13 categories)
