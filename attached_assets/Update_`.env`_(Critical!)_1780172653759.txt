### 1. Update `.env` (Critical!)
Add these **required** new variables to your `backend/.env`:
```bash
JWT_SECRET=your-very-long-secret-key-at-least-32-characters-long
FOUNDER_DEFAULT_PASSWORD=your-secure-founder-password
MONGO_URL=mongodb://localhost:27017
DB_NAME=fuelpro
MPESA_ENV=sandbox
MPESA_CONSUMER_KEY=your_consumer_key
MPESA_CONSUMER_SECRET=your_consumer_secret
MPESA_PASSKEY=your_passkey
MPESA_CALLBACK_BASE_URL=https://your-domain.com/api
REDIS_URL=redis://localhost:6379/0
```

### 2. Install New Dependency
```bash
cd backend
pip install pytz
# Add to requirements.txt:
echo "pytz>=2024.1" >> requirements.txt
```

```bash
git add -A
git commit -m "feat: Security overhaul + 3 new features + CI/CD

SECURITY:

```
